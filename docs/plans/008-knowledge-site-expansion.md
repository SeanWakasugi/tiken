# 008 ナレッジサイト拡張

## 目的

スライドだけでなく、補足記事、制作メモ、図解パターン、改善ログを蓄積できるナレッジサイトへ拡張する。

## 対象範囲

- `src/pages/docs/`
- `src/pages/patterns/`
- `src/pages/slides/`
- `docs/`

## 作業

1. 補足記事のURL設計を決める
2. スライドと補足記事を相互リンクする
3. 図解パターンページを一覧しやすくする
4. 制作メモまたは改善ログの置き場所を決める
5. トップページに最新コンテンツ枠を追加する

## 完了条件

- スライド、記事、図解パターンの導線がある
- 新しい知見をどこに追加すべきか迷わない
- `npm run build` が成功する

## 検証コマンド

```bash
npm run build
npm run dev
```

## 成果物

- ナレッジサイトとしての導線改善
- 補足記事の置き場所
- 改善ログ運用の方針
