# 005 初回サンプルスライドの完成度向上

## 目的

初回サンプルスライドを、tiken の考え方を説明できる実用サンプルとして完成させる。

## 対象範囲

- `src/pages/slides/ai-slide-workflow/index.astro`
- `src/pages/slides/ai-slide-workflow/script.md`
- `src/styles/slide.css`
- `docs/review-checklist.md`

## 作業

1. スライド構成を12枚前後に拡張する
2. 各スライドを1メッセージに絞る
3. 図解を2種類以上入れる
4. `script.md` を全スライド分そろえる
5. 公開前チェックリストで自己レビューする
6. PC幅とスマホ幅で表示確認する

## 完了条件

- サンプルスライドだけで tiken の目的と使い方が伝わる
- 全スライドに対応するスクリプトがある
- 図解とアニメーションが理解補助として機能している
- `npm run build` が成功する

## 検証コマンド

```bash
npm run build
npm run dev
```

## 成果物

- 完成度を上げた初回サンプルスライド
- 全スライド分の発表用スクリプト
- レビュー結果メモ
