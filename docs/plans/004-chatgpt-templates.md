# 004 ChatGPT制作テンプレートのサイト反映

## 目的

ChatGPTへの依頼テンプレートを、リポジトリ内の docs だけでなく公開サイトからも参照できる状態にする。

## 対象範囲

- `docs/chatgpt-workflow.md`
- `docs/templates/`
- `src/pages/docs/`

## 作業

1. 構成整理、実装依頼、レビュー依頼のテンプレートを `docs/templates/` に分割する
2. `/docs/` から主要テンプレートへアクセスできるページを作る
3. 各テンプレートに入力例と出力例を追加する
4. 公開してはいけない情報を貼らない注意書きを追加する
5. `README.md` からテンプレートへリンクする

## 完了条件

- docs と公開サイトの両方からテンプレートを探せる
- 依頼テンプレートがコピーペーストしやすい形になっている
- `npm run build` が成功する

## 検証コマンド

```bash
npm run build
```

## 成果物

- ChatGPT依頼テンプレート集
- 公開サイト上のテンプレート入口
